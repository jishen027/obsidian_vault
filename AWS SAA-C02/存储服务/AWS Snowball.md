**AWS Snowball** 是一种 PB（千万亿字节）级的数据传输方案，通过物理存储设备安全地将大量数据移入或移出 AWS

- **适用场景：** 当您的本地数据量巨大（如 25TB 以上），且互联网带宽受限导致网络传输耗时过长（例如 10MB/s 的带宽上传 1TB 数据需要约 10 天）时，Snowball 是最快且最经济的选择
- **工作流程**
	- 在 AWS 控制台申请设备
	- AWS 会寄给您一个物理硬件设备
	- 您在本地通过网络将数据拷贝到该设备中
	- 将设备寄回 AWS，由 AWS 将数据上传至您的 **S3 存储桶**
- **安全性：** 设备非常坚固，且内置 **256 位加密**保护，确保物理运输过程中的数据安全
